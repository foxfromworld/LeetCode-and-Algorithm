```python

class Solution(object):
  def asteroidCollision(self, asteroids):
    """
    :type asteroids: List[int]
    :rtype: List[int]
    """
    ret = []
    for ast in asteroids:
      while ret and ast < 0 < ret[-1]:
        if ret[-1] == -ast:
          ret.pop()
          break
        elif ret[-1] > -ast:
          break
        else:
          ret.pop()
          continue
      else:
        ret.append(ast)
    return ret
```
